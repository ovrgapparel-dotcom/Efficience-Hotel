# Guide d'Installation Totale - Efficience Hotel ERP v2.1 (Premium)

Ce pack contient tout le nécessaire pour déployer l'ERP en local dans un environnement client sécurisé.

## Contenu du Pack
1. `dist/` : Les fichiers compilés de l'application (React Native Web v2.0).
2. `Efficience_Hotel_Embed.html` : Un fichier autonome pour ouvrir l'ERP ou l'intégrer dans un intranet.
3. `serve_app.bat` : Script d'automatisation pour lancer le serveur local sur Windows.

## Pré-requis
- **Node.js** (Version 20 ou +) doit être installé sur le poste client.
- Navigateur moderne (Chrome, Edge, Safari).

## Instructions de Déploiement Local
1. Copiez le dossier `installation_pack` sur le poste cible (ex: sur le Bureau).
2. Lancez le fichier `serve_app.bat` par un double-clic.
3. Une fois le serveur lancé, ouvrez votre navigateur sur : `http://localhost:8080`
